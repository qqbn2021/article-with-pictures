如果文章没有缩略图，可以通过本插件自动给文章生成缩略图，同时支持在文章内容页显示缩略图。

== Description ==

支持文章自动生成缩略图，可以给文章设置默认缩略图。支持在文章内容页显示缩略图。

如果文章有缩略图，则会显示文章原缩略图。只有文章没有缩略图，才会显示插件自动获取的缩略图。

如果文章内容没有图片，可以配置是否在内容页显示缩略图，让文章图文并茂。

只有主题支持缩略图功能，才会显示缩略图。

1、支持设置文章配图接口，让配图更简单。

2、支持设置缩略图大小。

3、支持设置默认缩略图。如果缩略图生成失败，则会显示默认缩略图。

4、支持文章内容显示缩略图，可在文章首段、尾段、中间、随机显示。

== Installation ==

1. 进入WordPress网站后台，找到“插件-安装插件”菜单；
2. 点击界面左上方的“上传插件”按钮，选择本地提前下载好的插件压缩包文件（zip格式），点击“立即安装”；
3. 安装完成后，启用 “文章配图” 插件；
4. 通过“设置”链接进入插件设置界面；
5. 完成设置后，插件就安装完毕了。

== Frequently Asked Questions ==

= 使用文章配图后有什么效果？ =
让网站页面显示更友好，同时对收录有一定的帮助。

= 卸载插件后，是否会删除缩略图？ =
卸载插件后，除生成的缩略图不会删除，其余内容均会删除。

= 为什么文章配图插件无法正常使用？ =
如果您装的其它插件或者主题对缩略图也进行了处理，则文章配图插件很有可能无法正常运行。

= 商业版有哪些功能？ =
商业版内置了文章标题生成图片的功能，无需设置文章配图接口。
